# Vue (Scroll hide) Header
 
This is a small "renderless" Vue component which hide the header on scroll.

[Demo](https://antonnk.github.io/vue-scoll-hide-header/)

## How to 
```html
<body>
<scroll-header>
  <header>
    <nav> ... </nav>
  </header> 
</scroll-header>
<main>
  ...
</main>
</body>
```
